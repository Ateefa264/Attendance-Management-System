#Overview

A file-based C++ system to record employee attendance, handle different leave types (casual, earned, official, unpaid), manage approvals, maintain leave balances, and generate reports.

#Features

Mark and view attendance

Apply for leave

Supervisor/Director approval workflow

Track leave balance

Reports for low attendance, pending leaves, and employee details

#Technologies

C++ (OOP)

File handling (no database)
